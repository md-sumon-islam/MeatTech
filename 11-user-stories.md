# User Stories

1. **As a Registered Customer,** I want to select specific meat cuts (e.g., mince, cubes, steak boneless) so that I don't have to spend time cutting it at home.
2. **As a User,** I want to view the sourcing farm and slaughter date of the meat item so that I can be fully confident about its freshness.
3. **As a Delivery Driver,** I want an optimized delivery route on my app so that I can deliver the meat before the temperature rises.
4. **As an Admin,** I want to get low-stock alerts for popular meat products so that we can request immediate supply from affiliated farms.
