# User Journey Map

**User Objective:** Order 2kg of Premium Curry-Cut Beef for a family dinner.

1. **Discovery:** User opens MeatTech app -> Sees daily fresh stock banner.
2. **Customization:** Selects "Beef" -> Chooses "Curry Cut" -> Selects Fat Content level (Low Fat) -> Adds to cart.
3. **Checkout:** Enters delivery address -> Selects immediate delivery slot -> Pays via digital wallet.
4. **Waiting (Tracking):** Receives notification that the item is packed -> Tracks driver's bike on the map along with container temperature status (maintained at 3°C).
5. **Delivery:** Driver arrives -> Hands over clean, vacuum-sealed insulated package -> User scans QR code to verify meat origin -> Leaves a 5-star review.
